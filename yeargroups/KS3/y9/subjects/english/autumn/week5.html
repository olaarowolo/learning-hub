Learning Objectives

By the end of this week, you will be able to:

  • Construct complex sentences using subordinate clauses.
  • Use relative clauses to add detail and precision.
  • Combine ideas effectively using appropriate conjunctions.

Topic Introduction

Complex sentences allow writers to express sophisticated ideas by combining independent clauses with subordinate clauses. These sentences use subordinating conjunctions (although, because, when, if, etc.) and relative pronouns (who, which, that, etc.) to create more nuanced and detailed expressions. Mastering complex sentences improves writing clarity, sophistication, and the ability to show relationships between ideas.

In-Class Activities

  • Identify and analyse complex sentences in model texts.
  • Practice combining simple sentences into complex ones.
  • Create sentences using different subordinating conjunctions.
  • Use relative clauses to add descriptive detail.

Homework Assignments

  • Write 10 complex sentences using different subordinating conjunctions.
  • Include at least 3 sentences with relative clauses.
  • Ensure each sentence expresses a clear relationship between ideas.

Assessment Details

This week's assessment focuses on your ability to construct and use complex sentences effectively.

Assessment Questions

  1. Write five complex sentences using subordinating conjunctions.
  2. Write three sentences using relative clauses.
  3. Explain how complex sentences improve the clarity and sophistication of your writing.
